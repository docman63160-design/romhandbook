Base Hp +1200

Tai Chi runes

Convert the [0.2 / 5.0 / 12.0 / 22.5 / 35.0]% damage received to health loss, deducting the average over 3 seconds

Basic Hp +1200

Base Hp +1200

Spirit Rune

Recovers [10.0 / 250.0 / 600.0 / 1125.0 / 1750.0] Sp every 5 seconds, when consuming Sp there is a [0.2 / 5.0 / 12.0 / 22.5 / 35.0]% chance it won't be consumed

Base Hp +1200.

Attack, convert [0.04 / 1.0 / 2.4 / 4.5 / 7.0]% of the damage dealt into self-shield (shield value doesn't exceed the maximum health), lasting for 2 seconds. Base Hp +1200
